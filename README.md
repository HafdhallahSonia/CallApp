# 📱 Contact Manager - Application Flutter

Une application mobile de gestion de contacts développée avec Flutter et SQLite, permettant d'organiser vos contacts par catégories et de passer des appels directement depuis l'application.

## ✨ Fonctionnalités

- **Authentification** : Système de connexion et d'inscription sécurisé
- **Gestion des contacts** : Ajouter, modifier, supprimer et rechercher des contacts
- **Catégorisation** : Organiser les contacts par catégories personnalisées
- **Appels téléphoniques** : Lancer des appels directement depuis l'application
- **Interface intuitive** : Design moderne avec swipe pour éditer/supprimer
- **Base de données locale** : Stockage sécurisé avec SQLite
- **Recherche rapide** : Filtrer les contacts en temps réel

## 🛠️ Technologies utilisées

- **Flutter** : Framework de développement mobile
- **Dart** : Langage de programmation
- **SQLite (sqflite)** : Base de données locale
- **flutter_phone_direct_caller** : Gestion des appels téléphoniques
- **swipe_to** : Interactions swipe sur les cartes

## 📋 Prérequis

Avant de commencer, assurez-vous d'avoir installé :

- Flutter SDK (version 3.0 ou supérieure)
- Dart SDK
- Android Studio / Xcode (pour les émulateurs)
- Un éditeur de code (VS Code, Android Studio, etc.)

## 🚀 Utilisation

### Première utilisation

1. **Inscription** : Créez un compte avec votre email et mot de passe
2. **Connexion** : Connectez-vous avec vos identifiants
3. **Ajouter des catégories** : Créez des catégories pour organiser vos contacts
4. **Ajouter des contacts** : Ajoutez vos contacts avec nom, prénom, téléphone et catégorie


### Gestion des contacts

- **Swipe vers la droite** : Éditer un contact
- **Swipe vers la gauche** : Supprimer un contact
- **Icône téléphone** : Lancer un appel direct
- **Barre de recherche** : Filtrer les contacts par nom
